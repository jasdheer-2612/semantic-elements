Coding:

Coding is a process of identifying a passage in the text or other data items (photograph, image), searching and identifying concepts and finding relations between them. Therefore, coding is not just labeling; it is linking of data to the research idea and back to other data

Coding is a qualitative data analysis strategy that helps researchers: Define what the data is about Identify themes and the relationships between them Organize data so it can be examined and analyzed in a structured way Create analytic files Document and validate data across the research team
